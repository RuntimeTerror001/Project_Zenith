# Project Zenith

Project Zenith is a production-focused celestial exploration experience built with Next.js, TypeScript, Tailwind CSS, Framer Motion, Zustand, TanStack Query, Leaflet, CesiumJS, and satellite.js.

## Development

Copy `.env.example` to `.env.local`, add a NASA API key for higher request limits, then run:

```bash
npm install
npm run dev
```

Quality checks:

```bash
npm run typecheck
npm run lint
npm run build
```
